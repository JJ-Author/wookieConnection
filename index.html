wookieConnection

A simple to use Connection for Triplestores with REST Service

View project onGitHub

A simple to use Connection for Triplestores based on jdbc4sparql from https://code.google.com/p/jdbc4sparql/
It will also have a REST-API so you can use our REST Service and it doesn't matter which Triplestore is in the background



Current State (Release 0.0.1):

 The Connection is finished (of course there is more todo but the main Code is finished)
